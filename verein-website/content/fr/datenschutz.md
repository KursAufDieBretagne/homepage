Nous sommes ravis de l'intérêt que vous portez à notre initiative « Cap sur la Bretagne ». La protection des données revêt une importance particulière pour nous. L'utilisation de notre site Internet est en principe possible sans fournir aucune donnée à caractère personnel. Toutefois, si une personne concernée souhaite bénéficier de services particuliers via notre site Internet (par exemple, le formulaire de contact), le traitement de données à caractère personnel peut s'avérer nécessaire.

## 2. Hébergement du site web (GitHub Pages)

Notre site web est hébergé par le prestataire de services **GitHub Inc.** (88 Colin P Kelly Jr St, San Francisco, CA 94107, États-Unis) via le service « GitHub Pages ».

À chaque consultation de notre site web, le système GitHub collecte automatiquement des données et des informations provenant du système informatique de l'ordinateur qui consulte le site. Les données suivantes sont collectées :  

- Informations sur le type de navigateur et la version utilisée

- Le système d'exploitation de l'utilisateur

- L'adresse IP de l'utilisateur

- Date et heure de l'accès

Le stockage temporaire de l'adresse IP par le système est nécessaire pour permettre la livraison du site web à l'ordinateur de l'utilisateur. Ces données sont stockées dans les fichiers journaux de GitHub afin de garantir la sécurité et la stabilité du système. GitHub agit pour nous en tant que sous-traitant. Vous trouverez de plus amples informations sur la protection des données chez GitHub ici : https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement

## 3. Formulaire de contact (form.taxi)

Si vous nous envoyez des demandes via le formulaire de contact, vos coordonnées figurant dans le formulaire de demande, y compris les coordonnées que vous y avez indiquées (nom, adresse e-mail, texte du message), seront enregistrées chez nous afin de traiter la demande et en cas de questions complémentaires.

Pour le traitement technique du formulaire de contact, nous utilisons le service **Form.taxi** (form.taxi e.K., Xantener Str. 122, 50733 Cologne, Allemagne).
  
Lorsque vous envoyez le formulaire, vos données sont cryptées et transmises aux serveurs de form.taxi en Allemagne, où elles sont traitées, puis transmises sous forme d'e-mail.

Vous trouverez de plus amples informations dans la politique de confidentialité de form.taxi : [https://form.taxi/de/privacy](https://form.taxi/de/privacy)

## 4. Cookies et suivi

Ce site web n'utilise **aucun** cookie de suivi, outil d'analyse (tel que Google Analytics) ou plugin de réseau social qui enregistre votre comportement de navigation.

## 5. Vos droits

Vous avez à tout moment le droit :

- Conformément à l'art. 15 du RGPD, de demander **des informations** sur vos données à caractère personnel que nous traitons.

- Conformément à l'art. 16 du RGPD, d'exiger sans délai la **rectification** de vos données à caractère personnel inexactes ou incomplètes que nous avons enregistrées.

- Conformément à l'article 17 du RGPD, d'exiger la **suppression** de vos données à caractère personnel enregistrées chez nous.

- Conformément à l'article 7, paragraphe 3 du RGPD, de **révoquer à tout moment le consentement que vous nous avez donné**. (Cela concerne par exemple les données du formulaire de contact).

Si vous souhaitez exercer vos droits, il vous suffit d'envoyer un e-mail à l'adresse du responsable mentionnée ci-dessus.
